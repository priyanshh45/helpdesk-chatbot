# Outlook Troubleshooting

## Description

Microsoft Outlook crashes, freezes, opens slowly, fails to send or receive emails, or does not start.

## Possible Causes

- Corrupted Outlook profile
- Faulty add-ins
- Network connectivity issue
- Mail server issue
- Damaged Office installation
- Full mailbox

## Resolution

1. Restart Outlook.
2. Restart the computer.
3. Open Outlook in Safe Mode:
   `outlook.exe /safe`
4. Disable unnecessary add-ins.
5. Check mailbox storage.
6. Perform Office Quick Repair.
7. Create a new Outlook profile.
8. Verify mail server connectivity.
9. Confirm Outlook is working.

## Escalation

Escalate if:

- Exchange server is unavailable.
- Multiple users are affected.
- Mailbox corruption is suspected.
- Office repair fails.
