# Software Installation Request

## Purpose

Provide a standard process for installing approved software on company-managed devices.

---

## Scope

Applicable to all employees requesting authorized software.

---

## Prerequisites

- Approved request.
- User authorization.
- Administrator privileges.
- Valid software license.

---

## Procedure

### Step 1

Receive the software installation request.

---

### Step 2

Verify management approval if required.

---

### Step 3

Confirm the software is approved by the IT department.

---

### Step 4

Verify license availability.

---

### Step 5

Install the software.

---

### Step 6

Apply updates and patches.

---

### Step 7

Launch the application to verify successful installation.

---

### Step 8

Confirm with the user that the software is working correctly.

---

## Common Issues

- Insufficient permissions
- License unavailable
- Installation failure
- Compatibility issue
- Antivirus blocking installation

---

## Escalation

Escalate if:

- Software requires vendor support.
- License procurement is required.
- Compatibility issues exist.
- Installation repeatedly fails.

---

## Best Practices

- Install only approved software.
- Keep software updated.
- Remove unused applications.
- Document software versions.
