# Email Access Request

## Description

Procedure for creating a new corporate email account.

## Prerequisites

- Manager approval
- Employee account
- Microsoft 365 license

## Procedure

1. Verify request.
2. Verify approval.
3. Assign license.
4. Create mailbox.
5. Configure Outlook.
6. Test email.
7. Share login details.

## Escalation

Escalate if:

- License unavailable
- Mailbox creation fails
- Exchange issue
