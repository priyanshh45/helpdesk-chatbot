# Wi-Fi Troubleshooting

## Purpose

Resolve wireless network connectivity issues.

---

## Common Symptoms

- Cannot connect to Wi-Fi
- Frequent disconnections
- Slow internet
- Limited connectivity

---

## Troubleshooting Steps

1. Verify Wi-Fi is enabled.
2. Restart Wi-Fi adapter.
3. Restart the router if applicable.
4. Forget and reconnect to the network.
5. Run Windows Network Troubleshooter.
6. Update Wi-Fi drivers.
7. Test internet connectivity.

---

## Common Causes

- Weak signal
- Incorrect password
- Router issue
- Driver issue
- ISP outage

---

## Escalation

Escalate if:

- Multiple users are affected.
- Wireless access point is down.
- Router requires replacement.
