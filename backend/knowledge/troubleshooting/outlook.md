# Outlook Troubleshooting

## Purpose

Resolve common Microsoft Outlook issues.

---

## Common Symptoms

- Outlook crashes
- Cannot send emails
- Cannot receive emails
- Slow Outlook
- Profile errors

---

## Troubleshooting Steps

1. Restart Outlook.
2. Restart the computer.
3. Open Outlook in Safe Mode.
4. Disable unnecessary add-ins.
5. Repair Microsoft Office.
6. Create a new Outlook profile.
7. Verify internet connection.

---

## Escalation

Escalate if:

- Exchange server is unavailable.
- Multiple users are affected.
- Mailbox corruption is suspected.
