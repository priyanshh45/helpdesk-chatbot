# Printer Troubleshooting

## Purpose

Provide quick troubleshooting steps for common printer problems.

---

## Common Symptoms

- Printer offline
- Paper jam
- Print queue stuck
- Cannot print
- Poor print quality

---

## Troubleshooting Steps

1. Ensure printer is powered on.
2. Check network or USB cable.
3. Restart the printer.
4. Clear print queue.
5. Restart Print Spooler service.
6. Reinstall or update printer driver.
7. Print a test page.

---

## Escalation

Escalate if:

- Hardware fault exists.
- Printer cannot connect to network.
- Multiple users cannot print.
