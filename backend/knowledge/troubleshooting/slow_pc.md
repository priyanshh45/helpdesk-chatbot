# Slow Computer Troubleshooting

## Purpose

This guide helps diagnose and resolve slow computer performance issues.

---

## Common Symptoms

- Slow startup
- Applications take a long time to open
- System freezes
- High CPU usage
- High memory usage
- Slow internet despite good connection

---

## Troubleshooting Steps

1. Restart the computer.
2. Close unnecessary applications.
3. Check CPU and Memory usage using Task Manager.
4. Disable unnecessary startup applications.
5. Delete temporary files.
6. Ensure at least 20% free disk space.
7. Run Windows Update.
8. Scan the system using antivirus software.
9. Restart and verify performance.

---

## Common Causes

- Too many startup applications
- Low RAM
- Low disk space
- Malware
- Outdated Windows
- Background processes

---

## Escalation

Escalate if:

- Hardware failure is suspected.
- SSD/HDD is failing.
- Performance remains poor after troubleshooting.
