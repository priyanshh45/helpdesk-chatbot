# IT Help Desk Knowledge Base

## Overview

This knowledge base contains documentation used by the AI-powered IT Help Desk Chatbot.

It includes:

- ITIL concepts
- ITSM processes
- Incident SOPs
- Service Request SOPs
- Troubleshooting Guides
- Frequently Asked Questions
- Company Policies

The chatbot retrieves information from these documents to answer user questions accurately.
